proyecto zoo

Este es un programa para gestionar los animales de un zoológico usando archivos csv. es modular (está separado en main.py y utils.py).
que ocupa?

    Tener python 3 instalado.

    Los archivos zoo.csv y clases.csv en la misma carpeta que el código.

Como se usa?

Abres la terminal en la carpeta del proyecto y pones:
python main.py

Que funciones trae?

    listar por clasificación: te muestra las clases (mamíferos, aves, etc) y eliges una.

    listar por característica: te enseña todas las propiedades (pelo, plumas, vuela...) y filtras por la que quieras.

    agregar animal: te pregunta el nombre, la clase, cuántas patas tiene y luego se avienta todas las características una por una (0 para no, 1 para sí).

    guardado automático: cuando sales con la opción 4, guarda todo en el csv para que no se pierda nada.
